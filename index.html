Kelompok Studi Linux Universitas Riau ( KSL UR )

Tentang KSL Universitas Riau

Kelompok Studi Linux Universitas Riau adalah komunitas pengguna sistem operasi GNU/Linux dan penggiat sumber terbuka ( free open source software ) Beranggotakan Mahasiswa aktif/non-aktif, Civitas akademika, Alumni, dan masyarakat sistem operasi GNU/Linux dan penggiat sumber terbuka yang tinggal di sekitar Universitas Riau.

Sejarah KSL Universitas Riau

Keanggotaan

Keanggotaan bersifat bebas (free) artinya semua orang diperbolehkan masuk, ikut serta dalam setiap kegiatan Kelompok Studi Linux Universitas Riau tanpa ketentuan-ketentuan namun harus mengikuti ketentuan komunitas
